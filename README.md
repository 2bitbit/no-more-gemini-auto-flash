# No More Gemini Auto Flash

当 Google Gemini 自动切换到 Flash (快速) 模型时，弹出醒目的红色警告框，提醒用户切回 Pro 模型。

## 安装步骤

1. 安装 [Tampermonkey (油猴)](https://www.tampermonkey.net/) 浏览器扩展，或者同类型的脚本管理器。
2. 将本仓库中的 `no-more-gemini-auto-flash.user.js` 代码添加为新脚本，并保存。
3. 刷新 Gemini 页面即可生效。

## 常见问题排查 (Troubleshooting)

如果你发现脚本安装了却没有生效（没有弹窗报错警示框），请按照以下步骤排查：

### 1. 检查油猴图标角标（是否成功注入页面）
确保在常规 Web 标签页打开了 Gemini 网页版，并观察右上角的油猴插件图标是否有**红色数字下标**（比如 `1`）。
- **完全没有数字下标**：说明插件被限制在这个网站工作。
  - **重要原因**：请不要在 Edge/Chrome 的 **PWA 桌面独立应用窗口**中打开应用！默认情况下绝大部分浏览器限制了插件在独立应用窗口中运行。**请务必在常规的由标签页组成的浏览器窗口中打开**，然后看看插件能不能有数字角标。
  - **权限不足**：右击拓展图标 -> “此扩展可读取和更改站点数据” -> 确保这里是“在所有网站上”。
  - **设置没有打开**：确保油猴的插件管理中打开了：`Allow User Scripts` 或类似的权限总开关。
